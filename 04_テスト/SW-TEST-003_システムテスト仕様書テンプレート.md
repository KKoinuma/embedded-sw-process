### ST-010: 電源投入シーケンス確認

| 項目 | 内容 |
|---|---|
| **目的** | 電源投入から正常動作状態に至るシーケンスを確認 |
| **前提条件** | 全接続確認済み、システム電源 OFF 状態 |
| **試験手順** | 1. 電源投入 2. 起動ログを確認 3. 正常動作状態への遷移を確認 4. 初期出力値を確認 |
| **期待結果** | 10秒以内に NORMAL 状態へ遷移、アラームなし |
| **合否基準** | 起動時間 ≤ 10秒、エラーログなし |

### ST-030: 連続稼働試験

| 項目 | 内容 |
|---|---|
| **目的** | 24時間連続稼働における安定動作を確認 |
| **試験手順** | 1. 通常稼働状態にセット 2. 24時間放置 3. ログを収集・確認 |
| **合否基準** | 24時間中エラーなし、メモリリークなし、CPU 使用率上昇なし |
