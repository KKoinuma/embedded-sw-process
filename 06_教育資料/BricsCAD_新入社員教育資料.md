## 第1章　CADの基礎知識

### 1.1 CADとは何か

CAD（Computer-Aided Design）とは、コンピュータを使って図面を作成するソフトウェアの総称です。

#### 手書き図面との違い

| 項目 | 手書き図面 | CAD |
|------|-----------|-----|
| 修正 | 消して書き直し | 簡単に修正できる |
| 精度 | 人の手に依存 | 数値入力で正確 |
| 複製 | コピー機が必要 | ファイルをコピーするだけ |
| 管理 | 物理的な保管が必要 | デジタルで一元管理 |
| 共有 | 郵送・持参が必要 | メール・ネットワークで即時共有 |

#### CADを使うメリット

- **正確な寸法で作図できる**：数値を直接入力するため、ミリ以下の精度で描ける
- **修正が容易**：一部を変更しても全体を描き直す必要がない
- **再利用できる**：過去の図面を流用・改変して効率的に新しい図面を作れる
- **印刷品質が安定する**：何度印刷しても同じ品質が保たれる

---

### 1.2 BricsCADの概要と用途

BricsCADはベルギーのBricsys社が開発した2D/3D CADソフトウェアです。

#### 特徴

- AutoCAD（業界標準CAD）と高い互換性を持つ
- DWG形式（AutoCADの標準ファイル形式）をネイティブで扱える
- AutoCADと同様のコマンド体系を採用しており、操作感が近い
- AutoCADと比較してライセンスコストが低い

#### 主な用途

- 機械部品の設計図面
- 建築・土木図面
- 電気・配線図
- 製造現場での生産図面

---

### 1.3 ファイル形式（DWG）について

BricsCADで作成・保存するファイルの標準形式は **DWG形式**（拡張子：`.dwg`）です。

| 形式 | 説明 |
|------|------|
| `.dwg` | BricsCAD・AutoCADの標準形式。主にこちらを使用する |
| `.dxf` | 異なるCADソフト間でデータをやり取りする際に使う交換形式 |
| `.bak` | 保存時に自動作成されるバックアップファイル |

> **注意：** ファイルを保存する際、バージョンを選択できます。他社・取引先とデータをやり取りする場合は、相手の使用するバージョンに合わせて保存してください。

## 第2章　基本操作と画面構成

### 2.1 起動・終了・ファイル操作

#### BricsCADの起動

1. デスクトップのBricsCADアイコンをダブルクリックする
2. スタートアップダイアログが表示される
3. 「新しい図面を開始」または既存ファイルを選択する

#### ファイルの新規作成

- メニュー：**ファイル → 新規作成**
- ショートカット：`Ctrl + N`
- テンプレートの選択画面が表示されたら、社内標準テンプレートを選択する（第8章参照）

#### ファイルを開く

- メニュー：**ファイル → 開く**
- ショートカット：`Ctrl + O`

#### ファイルの保存

- 上書き保存：`Ctrl + S`
- 名前を付けて保存：`Ctrl + Shift + S`

> **重要：** こまめに保存する習慣をつけてください。`Ctrl + S` を作業中に定期的に押すことを推奨します。

#### BricsCADの終了

- メニュー：**ファイル → 終了**
- ショートカット：`Alt + F4`
- 未保存の変更がある場合は保存確認ダイアログが表示される

### 2.3 マウス操作

| 操作 | 動作 |
|------|------|
| 左クリック | 図形の選択・点の指定 |
| 右クリック | コンテキストメニューを表示 |
| ホイール回転（前） | ズームイン（拡大） |
| ホイール回転（後） | ズームアウト（縮小） |
| ホイールクリック＋ドラッグ | パン（画面をスクロール） |
| ホイールダブルクリック | 全体表示（ZOOM → Extents） |

> **ヒント：** ズームはマウスカーソルの位置を中心に拡大・縮小されます。見たい場所にカーソルを合わせてホイールを操作してください。

### 2.4 コマンドの入力方法

BricsCADでは主に3つの方法でコマンドを実行できます。

#### ① リボンアイコンをクリック

画面上部のリボンからアイコンを選択して実行します。初心者にわかりやすい方法です。

#### ② コマンドラインに直接入力

画面下部のコマンドラインに英語のコマンド名を入力して `Enter` を押します。

```
コマンド：LINE  ← 入力して Enter
```

よく使うコマンドを覚えると、リボンより素早く操作できます。

#### ③ ショートカットキー

よく使うコマンドには2〜3文字の短縮コマンド（エイリアス）があります。

| コマンド | 短縮 | 動作 |
|---------|------|------|
| LINE | L | 線分を描く |
| CIRCLE | C | 円を描く |
| ARC | A | 弧を描く |
| MOVE | M | 移動 |
| COPY | CO | コピー |
| ERASE | E | 削除 |
| ZOOM | Z | ズーム |

#### コマンドのキャンセル

実行中のコマンドをキャンセルするには `Esc` キーを押します。

## 第3章　作図の準備

### 3.1 単位と縮尺の考え方

#### 図面の単位

BricsCADでは、図面上の1単位が実際の寸法の1mmに対応するように設定して作図します（1:1で描く）。

- **正しいアプローチ：** 実際の寸法のまま入力する（例：100mmの線 → 100と入力）
- **間違いやすい点：** 縮尺を考えて入力する必要はない。縮尺は印刷時に設定する

#### 単位の設定方法

1. コマンドライン に `UNITS` と入力して `Enter`
2. 「図面単位」ダイアログが表示される
3. 長さの形式：**十進法**、精度：**0.00**（必要に応じて変更）
4. 挿入縮尺：**ミリメートル** を選択
5. `OK` をクリック

---

### 3.2 グリッドとスナップの設定

#### グリッド

描画エリアに表示される格子状の点または線です。目安として使いますが、グリッド上以外にも点を置けます。

- ON/OFF：ステータスバーの **[グリッド]** ボタン、またはキー `F7`

#### スナップ

カーソルが特定の間隔に「吸い付く」機能です。グリッドスナップをONにすると、設定した間隔の倍数の点にしか入力できなくなります。

- ON/OFF：ステータスバーの **[スナップ]** ボタン、またはキー `F9`

#### グリッド・スナップの間隔設定

1. ステータスバーの **[グリッド]** または **[スナップ]** を右クリック
2. 「作図設定」を選択
3. 「スナップとグリッド」タブで間隔を設定

---

### 3.3 オブジェクトスナップの種類と使い方

オブジェクトスナップ（Osnap）は、既存の図形上の特定の点（端点・中点・中心など）に正確にカーソルを合わせる機能です。

#### 主なオブジェクトスナップの種類

| スナップ名 | マーカー | 説明 |
|-----------|---------|------|
| 端点 (Endpoint) | □ | 線・弧の端点 |
| 中点 (Midpoint) | △ | 線・弧の中点 |
| 中心 (Center) | ○ | 円・弧の中心 |
| 交点 (Intersection) | × | 2つの図形の交点 |
| 垂線 (Perpendicular) | ⊥ | 図形に対して垂直な点 |
| 接点 (Tangent) | ○ | 円・弧への接点 |
| 近傍点 (Nearest) | × | 図形上の最も近い点 |

#### オブジェクトスナップの設定

1. ステータスバーの **[オスナップ]** を右クリック → 「オブジェクトスナップ設定」
2. よく使うスナップにチェックを入れる
3. ON/OFF：`F3` キー

> **ヒント：** オブジェクトスナップをONにしておくと、カーソルを図形に近づけた際に自動的にスナップポイントが表示されます。

---

### 3.4 レイヤーの概念（レイヤーとは何か・なぜ使うか）

レイヤーとは、図面を複数の透明なシートに分けて管理する仕組みです。

```
【イメージ図】
─────────────────── レイヤー：寸法線（青）
─────────────────── レイヤー：注釈・テキスト（緑）
─────────────────── レイヤー：中心線（赤・一点鎖線）
─────────────────── レイヤー：外形線（白・実線）
```

#### レイヤーを使うメリット

- 種類ごとに色や線種をまとめて設定できる
- 特定のレイヤーだけ表示・非表示にできる（例：印刷時に寸法を隠す）
- 特定のレイヤーをロックして誤編集を防げる

---

### 3.5 レイヤーの作成

#### レイヤーマネージャーを開く

- コマンドライン：`LAYER` と入力して `Enter`
- またはリボン：**ホーム タブ → レイヤー パネル → レイヤーマネージャー** アイコン

#### 新しいレイヤーを作成する手順

1. レイヤーマネージャーを開く
2. **「新規レイヤー」** ボタン（新規ページアイコン）をクリック
3. レイヤー名を入力する（例：`外形線`）
4. **色** の欄をクリックして色を選択する
5. **線種** の欄をクリックして線種を選択する
   - 実線（Continuous）：外形線など
   - 一点鎖線（CENTER）：中心線
   - 破線（DASHED）：隠れ線
6. **線の太さ** の欄をクリックして太さを設定する
7. `OK` をクリックして閉じる

> **注意：** 線種を選択する際に一覧に出てこない場合は「ロード」ボタンを押して線種ファイルを読み込む必要があります。

#### レイヤープロパティの設定項目

| 項目 | 説明 |
|------|------|
| 名前 | レイヤーの名前（半角英数字または日本語） |
| 表示（電球アイコン） | ON：表示 / OFF：非表示（印刷されない） |
| フリーズ（雪の結晶） | フリーズ中は表示・編集不可 |
| ロック（鍵アイコン） | ロック中は編集不可（表示はされる） |
| 色 | そのレイヤーに描いた図形のデフォルト色 |
| 線種 | そのレイヤーに描いた図形のデフォルト線種 |
| 線の太さ | そのレイヤーに描いた図形のデフォルト線幅 |
| 印刷 | 印刷対象にするかどうか |

---

### 3.6 レイヤーの切り替え・表示／非表示・ロック

#### 作図するレイヤーを切り替える

リボン上部のレイヤードロップダウンリストから、作図したいレイヤー名を選択します。  
選択したレイヤーが「現在のレイヤー」になり、以降に描く図形はそのレイヤーに属します。

```
[外形線 ▼]  ← クリックしてレイヤーを選択
```

#### 表示／非表示の切り替え

- レイヤーマネージャーの **電球アイコン** をクリックしてON/OFFを切り替える
- 非表示のレイヤーは画面にも印刷にも出力されない

#### レイヤーのロック

- レイヤーマネージャーの **鍵アイコン** をクリックしてロック/解除を切り替える
- ロックされたレイヤーの図形は選択・編集できない（誤操作防止に有効）

## 第4章　基本図形の作図

### 4.1 座標入力の方法

BricsCADで点を指定する方法は3種類あります。

#### ① 絶対座標

原点（0,0）からのX・Y距離を指定します。

```
書式：X,Y
例：100,50  → X=100, Y=50 の位置
```

#### ② 相対座標

**直前に指定した点** からのX・Y距離を指定します。実際の作図で最もよく使います。

```
書式：@X,Y
例：@100,0  → 直前の点からX方向に100移動した位置
```

#### ③ 極座標（相対）

直前の点からの **距離と角度** で指定します。斜め線を描くときに便利です。

```
書式：@距離<角度
例：@100<45  → 直前の点から距離100、角度45°の位置
```

角度は X軸正方向（右）を0°として、反時計回りが正です。

| 角度 | 方向 |
|------|------|
| 0° | 右（X正方向） |
| 90° | 上（Y正方向） |
| 180° | 左（X負方向） |
| 270° | 下（Y負方向） |

---

### 4.2 線分（LINE）

直線を描くコマンドです。連続して複数の線を描けます。

#### 操作手順

1. コマンドライン：`L` と入力して `Enter`（または リボン：ホーム → 線分）
2. 「1点目を指定：」→ 始点をクリックまたは座標入力
3. 「次の点を指定：」→ 終点をクリックまたは座標入力
4. 続けて入力すると連続した線分を描ける
5. `Enter` または `Esc` で終了

#### よく使うオプション

| 入力 | 動作 |
|------|------|
| `U` + Enter | 直前に描いた線分を取り消す |
| `C` + Enter | 最初の点に閉じて終了 |

---

### 4.3 ポリライン（PLINE）

複数の線分や弧を一つのオブジェクトとして描くコマンドです。  
LINEと異なり、全体が一つの図形として扱われます。

#### 操作手順

1. コマンドライン：`PL` と入力して `Enter`
2. 始点を指定
3. 次の点を指定（繰り返し）
4. `Enter` で終了

---

### 4.4 円（CIRCLE）

#### 操作手順（中心・半径で描く場合）

1. コマンドライン：`C` と入力して `Enter`
2. 「円の中心点を指定：」→ 中心をクリックまたは座標入力
3. 「半径を指定：」→ 半径の値を入力（例：`50`）して `Enter`

#### その他の描き方オプション

| オプション | 説明 |
|-----------|------|
| 直径(D) | 半径の代わりに直径で指定 |
| 2点(2P) | 直径の両端2点で指定 |
| 3点(3P) | 円周上の3点で指定 |
| 接線-接線-半径(T) | 2つの図形に接する円 |

---

### 4.5 円弧（ARC）

#### 操作手順（3点で描く場合）

1. コマンドライン：`A` と入力して `Enter`
2. 弧の始点を指定
3. 弧の通過点（中間点）を指定
4. 弧の終点を指定

#### よく使う描き方

| オプション | 説明 |
|-----------|------|
| 3点 | 始点・通過点・終点 |
| 始点・中心・終点 | 中心を指定して描く |
| 始点・中心・角度 | 中心と角度で描く |

---

### 4.6 矩形（RECTANG）

#### 操作手順

1. コマンドライン：`REC` と入力して `Enter`
2. 「1つ目のコーナーを指定：」→ 左下の角をクリック
3. 「反対のコーナーを指定：」→ 右上の角をクリックまたは相対座標入力
   - 例：`@100,50` → 幅100mm、高さ50mmの矩形

---

### 4.7 多角形（POLYGON）

正多角形を描くコマンドです。

#### 操作手順

1. コマンドライン：`POL` と入力して `Enter`
2. 辺の数を入力（例：`6` で正六角形）
3. 中心点を指定
4. 「内接円(I)」または「外接円(C)」を選択
5. 半径を入力

---

## 第5章　図形の編集

### 5.1 選択方法

コマンド実行前、または実行後に「オブジェクトを選択」と表示されたときの選択方法です。

#### ① 個別選択（クリック）

図形を直接クリックして選択します。複数選択する場合はクリックを繰り返します。

#### ② 窓選択（左から右へドラッグ）

**左から右へ** ドラッグすると青い実線の矩形が表示されます。  
矩形の **内側に完全に含まれる** 図形のみ選択されます。

#### ③ 交差選択（右から左へドラッグ）

**右から左へ** ドラッグすると緑の破線矩形が表示されます。  
矩形に **少しでも触れている** 図形がすべて選択されます。

#### 選択の解除

- `Esc` キーで選択をすべて解除
- 選択済みの図形をShiftキーを押しながらクリックすると個別に解除

---

### 5.2 移動（MOVE）

図形を別の場所に移動します。

#### 操作手順

1. コマンドライン：`M` と入力して `Enter`
2. 移動する図形を選択 → `Enter`
3. 「基点を指定：」→ 移動の基準となる点をクリック
4. 「目的点を指定：」→ 移動先の点をクリックまたは座標入力

---

### 5.3 コピー（COPY）

図形を複製します。

#### 操作手順

1. コマンドライン：`CO` と入力して `Enter`
2. コピーする図形を選択 → `Enter`
3. 基点を指定
4. 目的点を指定（`Enter`を押すまで連続コピーできる）

---

### 5.4 削除（ERASE）

図形を削除します。

#### 操作手順

- 方法1：図形を選択してから `Delete` キーを押す
- 方法2：コマンドライン：`E` → 図形を選択 → `Enter`

#### 直前の操作を元に戻す

`Ctrl + Z` で直前の操作をキャンセルできます。誤って削除した場合もすぐに復元できます。

---

### 5.5 トリム（TRIM）

境界線を基準に、はみ出した部分を切り取ります。

#### 操作手順

1. コマンドライン：`TR` と入力して `Enter`
2. 「切り取り境界を選択：」→ 境界にする図形を選択 → `Enter`
   （何も選択せず `Enter` を押すと全図形が境界になる）
3. 切り取りたい部分をクリック
4. `Enter` または `Esc` で終了

---

### 5.6 延長（EXTEND）

図形を境界線まで延長します。

#### 操作手順

1. コマンドライン：`EX` と入力して `Enter`
2. 「延長境界を選択：」→ 延長先の境界図形を選択 → `Enter`
3. 延長したい図形の境界に近い側をクリック
4. `Enter` または `Esc` で終了

---

### 5.7 オフセット（OFFSET）

図形を指定した距離だけ平行移動したコピーを作成します。平行線や同心円を描くときに使います。

#### 操作手順

1. コマンドライン：`O` と入力して `Enter`
2. 「オフセット距離を指定：」→ 距離を入力（例：`10`）→ `Enter`
3. オフセットする図形をクリック
4. オフセットする方向（内側・外側）をクリック
5. 繰り返し可能。`Enter` で終了

---

### 5.8 鏡像（MIRROR）

図形を指定した軸で反転コピーします。

#### 操作手順

1. コマンドライン：`MI` と入力して `Enter`
2. 鏡像にする図形を選択 → `Enter`
3. 鏡像軸の1点目を指定
4. 鏡像軸の2点目を指定
5. 「元のオブジェクトを削除しますか？」→ `N` (残す) または `Y` (削除)

---

### 5.9 配列複写（ARRAY）

図形を規則正しく複数コピーします。

#### 矩形配列（行・列）

1. コマンドライン：`AR` → `Enter`
2. 図形を選択 → `Enter`
3. 「矩形(R)」を選択
4. 行数・列数・行間隔・列間隔を設定

#### 円形配列（極座標配列）

1. コマンドライン：`AR` → `Enter`
2. 図形を選択 → `Enter`
3. 「極座標(PO)」を選択
4. 中心点・個数・角度を設定

---

## 第6章　注釈・寸法

### 6.1 テキストの入力と編集

#### 1行テキスト（DTEXT）

1. コマンドライン：`DT` → `Enter`
2. テキストの挿入点をクリック
3. 文字の高さを入力（例：`3.5`）→ `Enter`
4. 回転角度を入力（通常は `0`）→ `Enter`
5. テキストを入力 → `Enter` で改行、2回 `Enter` で終了

#### 複数行テキスト（MTEXT）

1. コマンドライン：`MT` → `Enter`
2. テキストボックスの対角2点をドラッグで指定
3. テキストエディタが開くので文字を入力
4. エディタ外をクリックまたは「閉じる」で確定

#### テキストの編集

- テキストをダブルクリックすると編集モードになる

---

### 6.2 寸法線の種類と入力方法

寸法を記入する前に、寸法を記入するレイヤーを「現在のレイヤー」に設定してください。

#### 水平・垂直寸法（DIMLINEAR）

1. コマンドライン：`DLI` → `Enter`
2. 「1点目の延長線の起点：」→ 寸法の始点をクリック（端点スナップを使用）
3. 「2点目の延長線の起点：」→ 寸法の終点をクリック
4. 「寸法線の位置を指定：」→ 寸法線を配置する位置をクリック

#### 直列寸法（DIMCONTINUE）

直前に作成した寸法から連続して寸法を追加します。

1. コマンドライン：`DCO` → `Enter`
2. 次の寸法の終点をクリック（繰り返し）
3. `Enter` で終了

#### 直径寸法（DIMDIAMETER）

1. コマンドライン：`DDI` → `Enter`
2. 円または弧をクリック
3. 寸法線の位置をクリック

#### 半径寸法（DIMRADIUS）

1. コマンドライン：`DRA` → `Enter`
2. 円または弧をクリック
3. 寸法線の位置をクリック

#### 角度寸法（DIMANGULAR）

1. コマンドライン：`DAN` → `Enter`
2. 1辺目の線をクリック
3. 2辺目の線をクリック
4. 寸法線の位置をクリック

---

### 6.3 引出線（QLEADER）

注記などを引き出して記入するときに使います。

#### 操作手順

1. コマンドライン：`LE` → `Enter`
2. 矢印の先端位置をクリック
3. 折れ点・テキストの位置をクリック
4. テキストを入力 → `Enter` で確定

---

## 第7章　レイアウトと印刷

### 7.1 モデル空間とペーパー空間の違い

BricsCADには2種類の作業空間があります。

| 項目 | モデル空間 | ペーパー空間（レイアウト） |
|------|-----------|------------------------|
| 用途 | 実寸で図形を描く場所 | 印刷レイアウトを設定する場所 |
| 縮尺 | 常に1:1（実寸） | ビューポートで縮尺を設定 |
| 切り替え | 画面下の「モデル」タブ | 画面下の「レイアウト」タブ |

#### 基本的なワークフロー

1. **モデル空間** で実寸（1:1）の図面を描く
2. **ペーパー空間** で用紙サイズと縮尺を設定して印刷する

---

### 7.2 ビューポートの設定

ペーパー空間（レイアウトタブ）では、モデル空間の図面を「ビューポート」という窓を通して表示します。

#### ビューポートを作成する

1. レイアウトタブに切り替える
2. リボン：**レイアウト → ビューポート** をクリック
3. 用紙上でビューポートの範囲をドラッグして作成

#### ビューポートの縮尺を設定する

1. ビューポートの枠をダブルクリックしてモデル空間に入る
2. ステータスバーの縮尺ドロップダウンから縮尺を選択（例：1:10、1:50）
3. ビューポート外をダブルクリックしてペーパー空間に戻る

> **注意：** ビューポートを選択してプロパティパネルからも縮尺を設定できます。「標準縮尺」の欄で変更してください。

---

### 7.3 印刷設定・PDF出力

#### 印刷ダイアログを開く

- メニュー：**ファイル → 印刷**
- ショートカット：`Ctrl + P`

#### 主な設定項目

| 項目 | 設定内容 |
|------|---------|
| プリンター/プロッター | 出力先のプリンターまたはPDF出力を選択 |
| 用紙サイズ | A4、A3 など |
| 印刷領域 | レイアウト（ペーパー空間使用時）または範囲・ウィンドウ |
| 印刷縮尺 | ペーパー空間使用時は「1:1」 |
| 印刷スタイルテーブル | 色と線幅の印刷設定ファイル（.ctbファイル） |

#### PDFとして出力する

1. 「プリンター/プロッター」で **PDF** を選択
2. 「ファイル名」に出力先のパスと名前を入力
3. `OK` をクリック

---

## 第8章　社内運用ルール

### 8.1 社内テンプレートの使い方

新しい図面を作成するときは、必ず社内標準テンプレートを使用してください。

#### テンプレートの使用方法

1. ファイル → 新規作成（`Ctrl + N`）
2. テンプレート選択ダイアログで社内テンプレートを選択する
   - 保存場所：（社内サーバーのパスを記載）
3. `OK` をクリック

テンプレートには以下が設定済みです：
- 社内標準レイヤー一式
- 表題欄（図枠）
- 文字スタイル・寸法スタイルの設定

---

### 8.2 社内標準レイヤーの一覧と用途説明

以下のレイヤー構成を標準として使用してください。

| レイヤー名 | 色 | 線種 | 線幅 | 用途 |
|-----------|----|----|------|------|
| 外形線 | 白（7） | 実線 | 0.5mm | 部品・構造物の外形 |
| 隠れ線 | 青（5） | 破線 | 0.25mm | 見えない部分の輪郭 |
| 中心線 | 赤（1） | 一点鎖線 | 0.13mm | 対称軸・円の中心 |
| 寸法線 | 緑（3） | 実線 | 0.13mm | 寸法・注釈 |
| 文字 | 緑（3） | 実線 | 標準 | テキスト注記 |
| 補助線 | マゼンタ（6） | 実線 | 0.13mm | 作図補助（印刷不要の場合は非表示） |
| 表題欄 | 白（7） | 実線 | 0.35mm | 図枠・表題欄 |

> **注意：** 図形のプロパティは色・線種ともに「**ByLayer**」の設定のままにしてください。レイヤーの設定を変えることで、すべての図形に一括反映されます。

---

### 8.3 レイヤーの命名規則（社内ルール）

標準レイヤー以外にレイヤーを追加する場合は、以下の命名規則に従ってください。

```
[分類コード]_[内容]
例：MECH_溶接線、ELEC_配線
```

- 分類コード：担当部門や図面種別を示す3〜4文字の英大文字
- 区切り：アンダースコア（_）
- 内容：日本語または英語で図形の種類を示す

---

### 8.4 ファイル命名規則・保存場所

#### ファイル命名規則

```
[図番]-[Rev番号]_[図面名].dwg
例：ABC-001-R01_外形図.dwg
```

| 項目 | ルール |
|------|-------|
| 図番 | 設計管理番号（例：ABC-001） |
| Rev番号 | 改訂番号（R01, R02, ...） |
| 図面名 | 内容を示す名称（日本語可） |
| 拡張子 | `.dwg`（DWG形式） |

#### 保存場所

| 状態 | 保存先 |
|------|-------|
| 作業中 | （ローカルまたは個人フォルダのパスを記載） |
| 正式登録 | （社内サーバーの正式格納フォルダのパスを記載） |
| 発行済み | （発行済みフォルダのパスを記載） |

> **重要：** 作業中のファイルは必ず定期的にバックアップを取ること。正式発行の際は必ず上長の確認を得てから登録フォルダに保存すること。

## 付録：よくあるトラブルと対処法

| 症状 | 原因 | 対処法 |
|------|------|-------|
| 描いた図形が見えない | レイヤーが非表示またはフリーズ | レイヤーマネージャーで表示を確認 |
| 図形が選択できない | レイヤーがロックされている | レイヤーのロックを解除する |
| 線種が実線で表示される | 線種尺度が小さすぎる | LTSCALEの値を調整する |
| コマンドが反応しない | 前のコマンドが実行中 | `Esc` を押してキャンセル |
| スナップが効かない | オブジェクトスナップがOFF | `F3` でオスナップをONにする |
| 印刷すると白紙になる | 印刷範囲の設定が間違っている | 「印刷領域」を「ウィンドウ」で再指定 |
| 文字が豆腐（□）になる | フォントが見つからない | 使用フォントを確認・変更する |
