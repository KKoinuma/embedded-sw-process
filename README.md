# 日本放射線エンジニアリング株式会社 東海事業所
## ソフトウェア開発標準プロセス文書管理リポジトリ

---

### 概要

本リポジトリは、日本放射線エンジニアリング株式会社 東海事業所における組込みソフトウェア開発の標準プロセス文書を管理するものです。

放射線関連機器・システムの開発においては、高い信頼性・安全性が求められます。本プロセス文書群は、IEC 62138 / IEC 61508 等の規格を参考に、当事業所の実情に合わせて策定しました。
