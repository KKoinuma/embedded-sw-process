## 重大度定義

| 重大度 | 定義 | 対応期限 |
|---|---|---|
| **Critical** | システム停止・安全機能喪失・データ消失 | 当日対応 |
| **High** | 主要機能が動作不能、回避策なし | 48時間以内 |
| **Medium** | 機能の一部が動作不能、回避策あり | 次スプリント |
| **Low** | 軽微な不具合・表示異常 | 計画的対応 |

## ステータス定義

| ステータス | 意味 |
|---|---|
| **Open** | 修正未着手 |
| **In Progress** | 修正作業中 |
| **Fixed** | 修正完了、確認待ち |
| **Verified** | 修正確認済み |
| **Closed** | クローズ |
| **Won't Fix** | 修正しないと判断 (理由を備考に記載) |
